
# DEMO 

This is simple project. It is a college project


## Installation

Install Demo

```bash
  git clone https://github.com/Kingask-dark/Demo
  cd Demo
  pip install -r requirements.txt
```
Run Demo

```bash
    cd Demo
    python manage.py runserver
```
