
# DEMO 

This is simple Django project.


## Installation

Install Demo

```bash
  git clone https://github.com/Kingask-dark/Demo
  cd Demo
  pip install -r requirements.txt
```
Run Demo

```bash
    cd Demo
    python manage.py runserver
```
